# Portfólio - Paulo Cidro

Portfólio pessoal construído para apresentar meus projetos e minha trajetória como Desenvolvedor Full Stack.

## Tecnologias Utilizadas

- **React**
- **TypeScript**
- **Vite**
- **CSS (Vanilla, Variáveis e CSS Modules/Globais)**
- **FontAwesome (Ícones)**

## Principais Funcionalidades

- Design moderno, com tema claro e escuro (Dark/Light mode).
- Navegação responsiva e fluida.
- Apresentação de projetos e tecnologias utilizadas.
- Links diretos para contato via e-mail, WhatsApp e redes profissionais.

## Como executar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/pcidro/portfolio.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd portfolio
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Comandos Disponíveis

- `npm run dev`: Inicia o servidor local.
- `npm run build`: Compila a aplicação para produção na pasta `dist`.
- `npm run lint`: Executa a verificação de linting do projeto.
- `npm run preview`: Visualiza localmente o build gerado.
